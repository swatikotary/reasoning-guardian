\# Dashboard Skill



\## What this skill does

Visualizes all engineering decisions in real-time.

Shows decision cards with reasoning, alternatives, confidence levels, and conflict alerts.



\## Features

\- Live polling every 5 seconds

\- Conflict detection and alerts

\- Statistics (total decisions, active, conflicts, avg confidence)

\- Color-coded status (active, overridden, rejected)

\- Responsive grid layout



\## Runs on

Port 3001 — `node dashboard/server.js`



\## Files

\- index.html — UI

\- app.js — Frontend logic

\- server.js — API backend serving from SQLite
