\# Query Agent Skill



\## What this skill does

Answers natural language questions about engineering decisions stored in decisions.db.

Returns clean plain English answers — no raw JSON, no technical noise.



\## Trigger phrases

\- "Why did we choose X?"

\- "What decisions affect Y?"

\- "What did we reject and why?"

\- "What happened with X?"

\- "Are we contradicting ourselves?"

\- "Show all decisions"



\## Returns

Plain English answer including: what was decided, why, alternatives rejected, 

current status, contributor name.
